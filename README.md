# chrono-race
App to track the time of several teams

Work in progress

## command line

* only after git clone to run the initial setup:
`npm run setup`

* afterwards only if you need to 'update' the npm and jspm dependencies:
`npm run update`

* for development:
`npm run test`
`npm run dev`

* for building:
`npm run build`
`npm run serveBuild`
